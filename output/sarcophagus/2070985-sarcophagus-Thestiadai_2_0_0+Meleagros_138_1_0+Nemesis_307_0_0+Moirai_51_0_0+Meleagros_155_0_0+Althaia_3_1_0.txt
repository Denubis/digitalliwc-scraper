	Name:	Althaia Erinys Meleagros Moirai Nemesis Thestiadai
	Grössenangabe:	H. 0,74. L. 2,05. T. 0,98.
	Beschreibung:	In der Mitte Althaia mit wehendem Mantel, die mit abgewandtem Blick das Holzscheit in das Feuer auf dem Altar hält (rechteckiger Blockaltar mit profilierter Deckplatte in Übereckansicht). Hinter dem Altar eine Erinye mit einer brennenden Fackel, links daneben eine Frau, die ihren l. Fuß auf ein Rad stellt. Sie hält eine Tafel und einen Griffel in der Hand, um das Los der Althaia und ihres Sohnes aufzuschreiben (Koch: Moire, Simon (LIMC): Nemesis). Auf den rechts anschließenden Szenen ist der Tod des Meleager und die Tötung der Thestiaden durch Meleager zu erkennen. 
	Bibliographie:	Robert, C., SarkRel III 2, 277 Taf. 91; Koch, G., SarkRel XII 6 Nr. 116 Taf. 107, 1.
	Kommentar:	1907 aus Slg. Borghese.
