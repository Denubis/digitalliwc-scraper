	Beschreibung:	Trois scènes se suivent: supplicatio (des barbares suppliant le général vainqueur), sacrifice, dextrarum iunctio.Scène de sacrifice (au centre): un garçon accomplit une libation versant le contenu d'une patère sur un thymiatérion.  Derrière celui-ci, un jeune musicien joue de la double flûte, accompagnant le sacrifice qui va s'accomplir à côté de lui: deux hommes barbus s'apprêtent à immoler un taureau que l'un d'eux, agenouillé, tient par une corne alors que l'autre a déjà levé la hache sacrificielle pour le tuer. A l'arrière-plan la façade d'un temple.
	Bibliographie:	Levi, A., Sculture greche e romane del Palazzo Ducale di Mantova (1931) 86 ss. n. 186 pl. 100; Sichtermann, H., AA 1974, 316 s.;Koch / Sichtermann, RömSark 100.
