	Beschreibung:	Devant une grotte, Mithra, un genou prenant appui sur l'échine du taureau blanc, l'autre jambe tendue, dresse la gueule de l'animal de la main g. pour planter le couteau, qu'il tient de la main dr., dans le cou de l'animal. Un chien mord l'animal tandis qu'un serpent ondule en-dessous. Un scorpion (difficilement visible) devrait mordre les testicules de la victime. De chaque côté, les deux assistants, Cautès et Cautopatès, sont représentés dans des proportions beaucoup plus petites. Au-dessus de chacun d'entre eux, Sol et Luna assistent à la scène. - Le scorpion et le serpent sont censés être des animaux maléfiques tentant d'annihiler les bienfaits découlant de la mise à mort du taureau alors que le chien est censé vouloir les contrecarrer en aidant Mithra à ce que le sang ait un effet bénéfique pour l'univers.
