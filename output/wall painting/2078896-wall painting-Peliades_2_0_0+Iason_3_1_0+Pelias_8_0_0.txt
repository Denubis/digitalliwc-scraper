	Name:	Iason Pelias Peliades
	Beschreibung:	Iason steht mit nur einer Sandale am rechten Fuß vor einem Opfertisch auf dem eine kleine Kanne steht; ein Opferdiener stellt gerade eine Patera darauf ab. Vor dem Tisch befindet sich eine Situla. Im Hintergrund befindet sich ein Tempel vor dem Pelias  zwischen seinen beiden Töchtern (Peliades) steht. Unterhalb des Tempels wird gerade ein bekränzter Stier für das Opfer für Poseidon von einem bekränzten Opferdiener herangeführt.
