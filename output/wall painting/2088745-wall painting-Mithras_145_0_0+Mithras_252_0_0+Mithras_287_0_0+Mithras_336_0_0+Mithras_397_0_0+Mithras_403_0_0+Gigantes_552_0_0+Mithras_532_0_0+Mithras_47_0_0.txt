	Name:	Mithras Gigantes
	Beschreibung:	Mithras killing the bull in a cave. Several stars on the mantle of Mithras. The bull's tail turns into ears. Scorpion, serpent and dog. On the left Cautes, on the right Cautopates. On the rock a raven. Above, on the left the bust of Sol. In the left side panel Mithras riding the bull, the birth of Mithras: Mithras emerging from a rock, holding torch and dagger. Then Saturnus, Gigantomachy. In the right side panel the water miracle, the alliance, the admission, Mithras taurophoros.
