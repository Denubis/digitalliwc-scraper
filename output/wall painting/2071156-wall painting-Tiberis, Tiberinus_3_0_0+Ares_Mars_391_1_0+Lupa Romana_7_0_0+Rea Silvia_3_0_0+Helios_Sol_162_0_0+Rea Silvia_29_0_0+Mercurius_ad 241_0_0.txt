	Name:	Lupa Romana Hermes/Mercurius Rea Silvia Tiberis, Tiberinus Aphrodite/Venus Helios/Sol
	Beschreibung:	"Origins of Rome" painting. Mars descending to Rhea Silvia. In a rocky landscape near the Tiber and the fig-tree is Mercurius pointing out for a female figure (Rhea Silvia?) the location by a cave of the Lupa Romana, facing to the right and suckling the twins. Above Mars Sol on a quadriga.
	Bibliographie:	Schefold, WP 88.Dawson, C.M., Romano-Campanian Mythological Landscape Painting (YaleClSt 9, 1944) 103-104 Nr. 46;PompPittMos III (1991) 1064-1067, 4-7.
