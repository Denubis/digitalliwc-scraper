	Beschreibung:	Iason steht mit nur einer Sandale am rechten Fuß vor einem Opfertisch auf dem eine kleine Kanne steht; ein Opferdiener stellt gerade eine patera darauf ab. Vor dem Tisch befindet sich eine Situla. Im Hintergrund befindet sich ein Tempel (hier kaum erkennbar, aber durch die Malerei in der Casa di Giasone rekonstruierbar(LIMC V Iason 3)) vor dem Pelias  zwischen seinen beiden Töchtern (Peliades) steht. Unterhalb des Tempels wird gerade ein bekränzter Stier für das Opfer für Poseidon von einem bekränzten Opferdiener herangeführt.
